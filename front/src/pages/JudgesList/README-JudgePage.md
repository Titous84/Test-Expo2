<h2>Fonctionnement</h2>
La section judge-page est la page de gstion des juges.

Les administrateurs peuvent modifier les informations nécessaires pour un juges en particulier.

La page est une simple liste de base pour afficher tous les juges actifs. Si le crayon est appuyé a la ligne du juge désiré, le mode edit commence.

Les tables utilisés par cette section sont :
- <b>users</b> : Les informations de bases des juges sont stockés dans cette table.
- <b>judge</b> : Les informations comme sa catégorie et son user_id sont utilisés dans cette table.
